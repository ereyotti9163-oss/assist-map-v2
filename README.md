# アシストマジック v2 (Assist Magic v2)

世界中のサイクリストのために設計された、シンプルで視認性に優れたWeb地図アプリケーション。
日常のライドから未知のルート探索まで、あなたの走りを強力にサポートします。

![Main View](https://raw.githubusercontent.com/ereyotti9163-oss/3-oss.github.io/main/1609.png)

## 🚴 主な機能

* **全世界対応サイクリングマップ**: `CyclOSM`を採用。世界中のサイクリングロードが青い線で強調表示されます。
* **地形を把握する等高線**: 山岳エリアや丘陵地に入ると自動的に等高線が表示され、ルートの起伏を直感的に把握できます。
* **スリムな検索ボックス**: 画面を遮らないコンパクトな設計。地名、駅名、スポット名から素早く目的地を探せます。
* **高精度GPS追従**: 右下のターゲットボタン（🎯）をタップするだけで、現在地を常に画面中央にキープします。
* **安定のノースアップ固定**: 地図の回転をあえて廃止し、常に北を上に固定。動作の軽快さと、確実な方位確認を実現しました。
* **スマート追従解除**: 地図を手動でドラッグすると、即座に自由閲覧モードへ切り替わります。

## 🛠️ 技術仕様

* **Map Engine**: [Leaflet.js](https://leafletjs.com/)
* **Map Tiles**: [CyclOSM](https://www.cyclosm.org/) (OpenStreetMap based)
* **Search API**: Nominatim API (Secure JSONP connection)

## 🚀 使い方

1.  GitHub PagesのURLを開きます。
2.  上部の「検索...」窓に目的地を入力し、🔍ボタンでジャンプ。
3.  右下の🎯ボタンを押し、自身の移動に合わせて地図を自動スクロール。
4.  勾配のきついエリアでは等高線を参考に、ペース配分を管理。
